# Ruby的第一个程序HelloWorld

## 代码示例

```ruby
hello.rb  # 文件名
puts "Hello World!"
```

## 程序的运行

```bash
ruby hello.rb   //在终端输出Hello World!
```

## 注释

### 单行注释

```ruby
# 我是注释
```

### 多行注释

```ruby
=begin
我喜欢中华牌绿豆沙，绿豆沙是我最喜欢的甜食
你们喜欢吗？
=end
puts "我喜欢中华牌绿豆沙"
```

### 之后全注释

```ruby
puts "我喜欢林志玲"
__END__
但是林志玲不喜欢我，真是好伤心啊
puts "我开始喜欢刘亦菲了"
```
