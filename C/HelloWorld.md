# 第一个C语言程序HelloWorld

## 代码示范1

```c
#include <stdio.h>
void main(){
    printf("Hello World!");
}
```

## 代码示范2

```c
#include <stdio.h>
int main(){
    printf("Hello World");
    return 0;
}
```

## Windows系统下的编译

```bash
gcc hello.c //产生一个可执行文件a.exe
a.exe.    //执行该文件,控制台输出Hello World!
```
