# 内存管理

## 作用域和生命周期

C语言变量的作用域分为：

- 代码块作用域
- 函数作用域
- 文件作用域

### 局部变量

#### 定义

大括号{ }内部定义的变量就是局部变量

#### 特点

- 只有执行到定义变量的这个语句，系统才会给这个变量分配空间
- 当离开大括号，非静态局部变量自动释放
- 局部变量的作用域在当前的{ }，离开此大括号，无法使用此变量
- 内部的作用域可以使用外部作用域的变量
- 普通局部变量也叫自动变量
- 如果不初始化，默认值为随机数
- { }内部的普通局部变量，加不加auto关键字等价
- 就近原则：同名变量执行就近原则
- 不同的大括号中，变量名可以一样

### 静态static局部变量

#### 定义

大括号内部定义的static变量就是静态局部变量

#### 特点

- static局部变量，在编译阶段就会分配空间，函数没有调用前就已经存在
- 离开大括号，static局部变量不会释放空间，只有程序结束，static变量才会自动释放
- static局部变量的作用于在当前大括号，离开当前大括号，无法使用该变量
- 不同的大括号中，static局部变量名可以一样。
- 如果static局部变量不初始化，它的值默认为0
- static局部变量初始化语句，只会执行一次，但是可以赋值多次。
- static局部变量放在data区
- static局部变量只能用常量初始化

### 普通局部变量和静态局部变量区别

1. 内存分配和释放
   - 普通局部变量只有执行到定义变量的语句才分配空间
   - 静态局部变量在编译阶段（函数还没有执行），变量的空间已经分配
   - 普通局部变量离开作用域后，自动释放
   - 静态局部变量只有在整个程序结束后才会释放
2. 初始化
   - 普通局部变量不初始化，值为随机数
   - 静态局部变量不初始化，值为0
   - 静态局部变量初始化语句只执行第一次
   - 静态局部变量只能用常量初始化

### 全局变量

#### 定义

在函数外面定义的变量为全局变量

#### 特点

- 定义了全局变量，任何地方都能使用
- 如果使用变量时，前面找不到此全局变量的定义，需要声明后才能使用
- 声明只是针对全局变量，不是针对局部变量
- 全局变量只能定义一次，可以声明多次
- 全局变量不初始化，默认赋值0
- 全局变量在编译阶段已经分配空间（函数没有执行前），只有在整个程序结束，才自动释放

```c
#include <stdio.h>
/*
函数的声明
使用fun函数时，前面找不到fun函数的定义，需要先声明才能使用
extern可以不写
可以多次声明
*/
extern void fun();

int main(){
	fun();
	return 0;
}

void fun(){
	/*
	全局变量的声明
	声明时，不要赋值
	使用a变量时，前面找不到a的定义，需要声明才能使用
	可以多次声明
	*/
	extern int a;
	printf("a = %d\n",a);
}

int a = 10;
```

#### 全局变量的缺陷

- 只有声明，没有定义，无法给变量赋值
- 如果定义一个全局变量，没有赋值（初始化），无法确定是定义还是声明
- 如果定义一个全局变量，同时初始化，一定是定义
- 定义一个全局变量，建议同时初始化
- 如果声明一个全局变量，建议加extern关键字

#### 全局变量分文件

```c
//main.c文件
#include <stdio.h>

int main(){

```

### 静态static全局变量

- static全局变量和普通全局变量的区别就是作用域不一样，static全局变量的作用域是文件
- static全局变量，不做初始化，默认赋值0；
- static全局变量只能在当前文件使用，别的文件不能使用
- 普通全局变量，在声明后，所有文件都能使用
- 不同文件只能出现一个同名普通全局变量的定义
- 不同文件可以拥有各自的同名static全局变量的定义，各自独立
- extern关键字只适用于普通全局变量

### 全局函数和静态static函数

- 所有文件只能有一次普通函数的定义
- 一个文件可以有一个静态函数的定义
- 普通全局函数，声明后，所有文件都能调用
- 静态函数只能在定义所在的文件中使用
- 就近原则 

### 总结

|      类型      |   作用域   |    生命周期    |
| :------------: | :--------: | :------------: |
|    auto变量    | 一对{ } 内 |    当前函数    |
| static局部变量 | 一对{ }内  | 整个程序运行期 |
|   extern变量   |  整个程序  | 整个程序运行期 |
| static全局变量 |  当前文件  | 整个程序运行期 |
|   extern函数   |  整个程序  | 整个程序运行期 |
|   static函数   |  当前文件  | 整个程序运行期 |
|  register变量  | 一对{ }内  |    当前函数    |

## 内存布局

### 内存分区

#### text：代码区 

只读，函数，程序没有执行前已经确定

#### data：初始化的数据

全局变量，static变量，文字常量区（只读），程序没有执行前已经确定

#### bss：没有初始化的数据

全局变量，static变量，程序没有执行前已经确定

#### stack：栈区

普通局部变量，自动分配空间，自动释放，自动管理内存，先进后出

#### heap：堆区

手动申请空间，手动释放，整个程序结束，系统也会自动回收

如果没有手动释放，程序也没有结束，不会自动释放

### 内存操作函数

#### memset()

```c
#include <stdio.h>
#include <string.h>

int main(){
	int a;
	//清零功能
	memset(&a,0,sizeof(a));
	printf("a = %c\n",a); // a = 0
	/*
	中间参数虽然是整形，但是以字符处理
	*/
	memset(&a,65,sizeof(a));
	printf("a = %c\n",a); // a = A

	char str[10];
	memset(str,'a',sizeof(str));
	for(int i = 0;i<10;i++){
		printf("%d,",str[i]); //输出ascii码
		printf("%c,",str[i]); //输出字符
	}
	printf("\n");
	return 0;
}
```

#### memcpy()

```c
#include <stdio.h>
#include <string.h>

int main(){
	char p[] = "hello\0mike";
	char buf[11];
	printf("sizeof(p) = %lu\n",sizeof(p)); // sizeof(p) = 11
	strncpy(buf,p,sizeof(p));  	//遇到\0拷贝结束
	printf("buf = %s\n",buf); // buf = hello
	printf("buf = %s\n",buf + strlen("hello") + 1);//buf = 
	memset(buf,0,sizeof(buf));	//把buf每个元素清零
	/*
	拷贝指定的长度，不会因为结束符结束
	第三个参数是指拷贝内存的总大小，不是个数
	最好不要出现内存重叠
	如果出现内存重叠，最好使用memmove函数
	*/
	memcpy(buf,p,sizeof(p));
	printf("buf = %s\n",buf); // buf = hello
	printf("buf = %s\n",buf + strlen("hello") + 1);//buf = mike
	return 0;
}
```

#### memcmp()

```c
#include <stdio.h>
#include <string.h>

int main(){
	int p[5] = {1,2,3,4,5};
	int s[5] = {1,2,3,4,5};
    /*
    比较的原则是：只要有一个大，则大，不管后面的大小
    字符串的比较是比较ASCII
    */
	int flag = memcmp(p,s,5 * sizeof(int));
	if(flag == 0){
		printf("p == s\n");
	}else if(flag > 0){
		printf("p > s\n");
	}else{
		printf("p < s\n");
	}
}
```

### 堆区内存分配和释放

```c
#include <stdio.h>
#include <stdlib.h>

int main(){
	int *p;
	/*
	1、参数是指定堆区分配多大的空间
	2、返回值：
		- 成功就是堆区空间首元素地址
		- 失败则返回NULL
	3、动态分配的空间，如果程序没有结束，不会自动释放
	4、使用完，需要人为释放free(p)
	5、free(p)不是释放p变量，是释放p所指向的内存
	6、同一块堆区内存，只能释放一次
	7、所谓的释放不是指内存消失，是指这块内存用户不能再使用，由系统回收了；
	8、如果用户再用，就是操作非法内存
	*/
	p = (int *)malloc(sizeof(int));//堆区分配空间
	if(p == NULL){
		printf("分配空间失败\n");
		return -1;
	}
	*p = 10;
	printf("*p = %d\n",*p);
	free(p); //释放堆区内存
	return 0;
}
```
